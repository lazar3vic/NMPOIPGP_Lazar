# Contributing to NMPOIPGP_Lazar

Thank you for your interest in contributing! Here's how you can help improve this project.

## How to Contribute

### Reporting Bugs

- Check if the bug has already been reported in [Issues](../../issues)
- If not, create a new issue with a clear title and description
- Include steps to reproduce the bug
- Provide screenshots or error messages if applicable

### Suggesting Enhancements

- Open an issue with the label `enhancement`
- Describe the enhancement and why it would be useful
- Include examples of how it would work

### Submitting Code Changes

1. **Fork the repository** to your GitHub account
2. **Create a feature branch** from `main`:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** with clear, descriptive commit messages
4. **Test your changes** thoroughly
5. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request** with a clear description of your changes

### Pull Request Guidelines

- Reference any related issues (e.g., "Closes #123")
- Provide a clear description of what changed and why
- Ensure your code follows the project's style guidelines
- Include any necessary documentation updates
- Test your changes before submitting

## Code Style

- Follow the existing code conventions in the project
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions focused and modular

## Questions?

If you have questions, feel free to open an issue or reach out. We're here to help!
